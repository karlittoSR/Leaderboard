## 自动方法 (v1.1.0+)

### 首次使用
1. **双击** `get_game_id.ps1`
2. **主菜单** 自动显示
3. **导航** 使用↑↓选择"添加新预设"
4. **按Enter确认**
5. **输入** 游戏名称（例如："Elden Ring"）
6. **导航** ↑↓和**Enter**从列表中选择游戏
7. **导航** ↑↓和**Enter**选择类别（Any%、100%等）
8. **可选**：如果可用，选择子类别
9. **输入预设ID**（自动提供建议）
10. **自动激活** 预设（如果是第一个）

**结果**：预设已保存 + 自动激活 + URL已复制！

### 🌐 多语言支持 (v1.1.0)
界面支持**5种语言**：
- **🇫🇷 Français**（默认）
- **🇺🇸 English**
- **🇪🇸 Español**
- **🇧🇷 Português**
- **🇨🇳 中文**

**更改语言**：
1. 主菜单 → **选项5** "语言设置"
2. 使用↑↓导航选择您的语言
3. 按**Enter** → 立即更改！
4. 语言**自动保存**

**导航**：使用**↑↓**在选项间移动  
**选择**：按**Enter**确认  
**取消**：按**Esc**（屏幕显示时可用）

### 可用选项

**1. 添加新预设**：创建新预设（与首次使用相同的流程）  
**2. 查看详情**：查看预设的完整详情  
**3. 更改活动预设**：选择在OBS中活动的预设  
**4. 删除预设**：删除预设（需要确认）  
**5. 语言设置**：更改界面语言（5种语言可用）  
**6. 退出**：关闭程序

### 在OBS中使用
- **简单URL**：`leaderboard.html`（始终相同！）
- **浏览器源**：宽度400，高度280
- **自动刷新**：每30秒
- **无参数**：自动读取活动预设

## 常见问题

### PowerShell脚本被阻止
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 浏览器测试时出现"Failed to fetch"
**正常现象**：CORS限制。**在OBS中工作正常！**

### 预设未显示
1. 使用 [get_game_id.ps1](get_game_id.ps1) → **选项3** 更改活动预设
2. 所有设置由脚本自动管理

## 需要帮助？

- **找不到游戏ID** → 运行 `get_game_id.ps1` - 完全自动化！
- **没有跑步显示** → 使用脚本重新配置预设
- **轮播不滚动** → 跑步数量少于配置的数量
- **双击时"Failed to fetch"** → 运行本地服务器：`python -m http.server 8000`
- **PS脚本不执行** → 右键点击 → "用PowerShell运行"
- **更改语言** → 主脚本菜单中的选项5
